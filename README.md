# trigger_Evercam
The firmware and simple script with GUI for the selfmade trigger for the camera Evercam. Arduino sends the commands to the pins of the camera.

1) Включить камеру. Подсоединить триггер.
2) Запустить ПО VS CAM.
3) Отметить Триггер. Правой кнопкой мыши при нажатии на слово "Триггер" выбрать мультитриггер. Эта опция позволит писать кадры в память по порядку, не перезаписывая предыдущие. В ПО Evercam выбрать кол-во кадров, которые необходимо записывать в одной серии. Например, с "1 по 2" означает запись двух кадров в одной импульсе.
4) Открыть программу evercam_trigger.py для задания параметров триггера. Подключить порт, указать кол-во импульсов и паузу между ними. ВАЖНО: убедитесь, что запущена только одна программа, которая слушает указанный порт. Нажать кнопку "Send Command". ВАЖНО: в самом ПО EverCam необязательно выбирать LiveRegime c камеры. Счетчик кадров в ПО работает НЕ корректно. То есть при запуске триггера камера запишет нужные кадры, но счетчик может не сработать, поэтому на него ориаентироваться нельзя.

При запуске ПО EverCam независимо от того, выбрана опция Мультитриггер или нет, камера начнет писать с первого кадра. Если опция "Мультитриггер" при отмеченном "Триггер" не выбрана, камера всегда будет писать сначала при новом запуске. Если опция выбрана, то она продолжит писать кадры подряд.
